Network Growth Visualization

This code creates an interactive visualization of a growing network. It uses p5.js for graphics and Matter.js for physics simulation.

The network consists of circles (vertices) connected by lines. Here's how it works:

  • Clicking the mouse adds a new circle at that location.
  • Every few frames, the network grows automatically:
    • It selects an existing circle
    • Creates a new circle nearby
    • Connects them with a line
  • The network moves dynamically due to simulated physics
  • Circles with only one connection are drawn slightly larger
  • Pressing the spacebar pauses or resumes the animation

The code handles drawing, growth logic, and physics interactions. It creates a dynamic, expanding network structure that responds to user input and evolves over time.
